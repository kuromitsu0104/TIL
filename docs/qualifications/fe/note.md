    - IP アドレスとサブネットマスク
      - xxx.xxx.xxx.0/yy
        - xxx.xxx.xxx.0
          - IP アドレスのこと
        - /yy
          - サブネットマスクを CIDR 表記で表したもの
          - IP アドレスのネットワーク部, ホスト部を表す
      - example
        - 255.255.255.0/24
          - IP アドレス
            - 255.255.255.0
          - サブネットマスク
            - CIDR 表示
              - /24
            - 2 進表記
              - 11111111111111111111111100000000
      - note
        - ホスト部の IP アドレスはどちらの値「0 / 1」でも許容する
